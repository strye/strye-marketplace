# Feature Requirements Worker

This Skill creates or updates a feature's requirements.md file. It will use a prompt, a file reference, or probing questions to complete requirements.md

## Instructions
1. Use Read to view file contents
2. Use Grep to search within files
3. Use Glob to find files by pattern
